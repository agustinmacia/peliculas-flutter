# Aplicación de Películas

Aplicacion de reseñas de películas. La idea es poder orientalar a cines con sus carteleras.

## App

La idea es esta app es poder ofrecer el servicio de compra de entradas para las películas que se estrenarán en los cines. Se podrá reservar, elegir fecha y asientos.
